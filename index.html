switcher-js

Simulate your own on and off switch button with this simple plugin.

Usage

Html

Simply add an radio button on your page like this: <input type="radio" name="testing" id="switch" />

Javascript

And then call the plugin in your <head>: $(document).ready(function(){ $("#switch").switcher(); })

Tips

  1. It works like a real radio button, when it's turned off, it doesn't returns value on a submitted form.
